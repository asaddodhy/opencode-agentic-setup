## Calling Perplexity for generation

For complex coding tasks, call the Perplexity API for code generation:

```bash
curl -s http://localhost:8002/v1/chat/completions \
  -H "Authorization: Bearer $PERPLEXITY_SESSION_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "perp/anthropic/claude-sonnet-4.6",
    "messages": [{"role": "user", "content": "..."}],
    "search_focus": "writing"
  }'
```

**Model selection:**
- General coding: `perp/anthropic/claude-sonnet-4.6`
- Complex logic: `perp/anthropic/claude-opus-4.7-thinking` or `perp/openai/gpt-5.5-thinking`
- Simple tasks: `perp/openai/gpt-5.4`

Default `search_focus` is `"writing"` (pure generation). Use `"web"` only when you need live technical information.
